🔐 Smart Password Generator

A fast and optimized Password Generator built with ReactJS.
It’s a simple tool, but packed with smart React features and smooth UI/UX.


🚀 Features

Auto-generate password in real-time
Set password length (6 to 100)
Toggle numbers and special characters
Copy to clipboard with visual feedback
Fully client-side, no reloads
Fast and responsive


⚙️ Tech Used

ReactJS (Frontend framework)
JavaScript (Core logic)
React Hooks (useState, useEffect, useCallback, useRef) 
CSS / Tailwind (Styling and responsivenesst)
GitHub Pages  (Deployment)
