💱 Currency Converter — ReactJS


A dynamic and real-time Currency Converter built with ReactJS, using live exchange rates from a powerful API.
No hardcoded values — just fast, accurate conversions with smooth performance.

🚀 Key Features

🔁 Real-time currency conversion using live API
🔄 Swap functionality to switch "From" and "To" currencies instantly
🧠 Custom hook (useCurrencyInfo) to fetch and manage exchange rate data
📦 Component reusability for clean and maintainable code
📜 No hardcoded data — all currency info is fetched via API
⚡ Optimized performance even with large lists
🧩 Responsive design with smooth scroll for dropdowns


🧠 What I Learned

How to fetch and use live API data for real-world applications
Building and using a custom React hook (useCurrencyInfo)
Managing efficient re-renders and scrollable UI for long lists
Implementing swap logic in a clean and reusable way
Creating modular components like InputBox, App.jsx, etc.
